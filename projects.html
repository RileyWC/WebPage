Spam Email Filter Using Bayes' Theorem (Fall 2023)

For a Machine Learning class assignment I was instructed to make a Bayesian filter to detect spam emails from normal emails. This worked by tokenizing each word in a given email, and then checking a dataset of spam and clean emails for the amount of times the word was used in each category. Bayes' Theorem could then be used to calculate the probability that the email is spam by looking at the prevalence of all words in both categories.

As an example: if an email has the phrase "Click this link", each word ("Click", "This", "Link") will be searched for in the email database and there will be a count for each time the word appears in the spam list, as well as in the clean email list. This will be done for each word until every word has an associated probability of spam. ("Click": 67%, "This": 37%, "Link": 82%) After scanning every word in the email, if the probability of spam is above a certain threshold, the email will be marked as spam.

Author Style Recreation Using Markov Chains (Spring 2023)

This project took a large text file as input and would output a specified amount of characters into an output file in the likeness of the input text. This was done using markov chains, which look at the last word outputted (prefix) and searches the input text to see what words followed that input word (suffix). It would then choose a word to output by randomly choosing a suffix, weighted by how many times a suffix occurred in the input.

As an example: The program reads every word in the input text and marks how often that word appears. It then picks a word from that list randomly, but words that appear more often are more likely to be chosen. Now that a word (prefix) has been chosen and outputted, that word is searched for in the input text and every word that follows (suffix) is marked by how often it appears. Just like before, a word is chosen randomly, weighted by how often it appears in the input text. This continues word after word until a desired word count is met.
